# Rndr - PBR rendering engine

## Description

3D Render engine based on The Cherno's Game Engine Series

## Installation

```bash 
➜  ~ git clone --recursive https://github.com/imisumi/rndr-engine.git
➜  ~ cd rndr-engine
➜  ~ ./setup.bat "// Windows"
➜  ~ ./setup.sh "// Linux"
```

## Usage

Run the executable located in bin/${OS}/Debug/Sandbox.exe
